# Engfreq
#### 用这个应用你可以排名英文的20.000最常用的词语 
#### 下载 engfreq apk ： https://github.com/TMPKKecske/EngFreq/releases/download/1.1.0/engfreq-release.apk
## 教学：
1. 写你的生词在“Your words: ”
2. 打“Start ranking”
3. 你写的词语会变成排名的

## 应用特征：
- 突出了从第一名倒第三千五白名的生词
- 如果你关闭应用，你记下的词语会自动保存
